HTML Formatting Elements

Formatting elements were designed to display special types of text:

b= - Bold text

strong= - Important text

i= - Italic text The tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

em= - Emphasized text

mark= - Marked text

small= - Smaller text

del=- Deleted text

ins= - Inserted text

sub= - Subscript text

sup= - Superscript text
